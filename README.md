# JomBoss Mastermind

Internal operating brain for JomBoss Joss Sdn. Bhd.

This workspace is organised around three jobs:

- Maintain a reliable company source of truth.
- Keep the investor data room diligence-ready.
- Run a weekly CEO/PA review that records decisions, issues, and missing evidence.

## Working Rules

- `01_COMPANY/CURRENT_TRUTH.md` is the default source of truth.
- `00_SYSTEM/JOMBOSS_MASTER_PROTOCOL.md` defines how Codex should reason and respond.
- `10_INVESTOR_DATA_ROOM/DATA_ROOM_INDEX.md` is the navigation file for investor data room work.
- Missing, outdated, or unsupported claims should be tracked in `10_INVESTOR_DATA_ROOM/DATA_ROOM_GAP_TRACKER.md`.
- Major decisions belong in `00_SYSTEM/DECISION_LOG.md`.
- Unresolved risks belong in `00_SYSTEM/OPEN_ISSUES.md`.

## Source PDFs

Original PDFs and machine-extracted text are stored in `00_SYSTEM/00_SOURCE_PDFS/`.
