# Document Classification Rules

Use one classification label on every important file.

| Label | Meaning |
| --- | --- |
| public | Safe to share publicly |
| investor_safe | Safe for investors and accelerators |
| client_safe | Safe to share with clients |
| internal | Internal company use only |
| restricted_internal | Sensitive internal strategy, pricing, payout, SOP, or risk logic |
| legal_sensitive | Contracts, legal risk memos, regulatory analysis |
| financial_sensitive | Bank statements, revenue records, payout records, financial model |
| personal_data | Personal data of founders, talents, clients, or team members |
