# Version Control Rules

## Rules

- Current working documents should live in the main folder for their area.
- Old versions should be moved to an archive folder only when an archive folder exists for that area.
- Do not treat old pitch decks, brainstorming notes, or informal chat notes as current facts.
- When facts conflict, update `00_SYSTEM/OPEN_ISSUES.md` and resolve against `01_COMPANY/CURRENT_TRUTH.md`.
- Add meaningful dates to files that are periodic, time-sensitive, or evidence-based.
