# Task Brief

Review and approve the standard client-facing credit wording so credits are consistently described as service entitlements, not cash, stored value, wallet balance, e-money, withdrawable value, or transferable monetary value.

# Why This Matters

Inconsistent credit wording could create client disputes, refund confusion, accounting ambiguity, or legal/commercial risk. Quotations, invoices, service agreements, sales scripts, website pages, and future platform UI need consistent language.

# Current Progress

The credit system policy and legal position memo exist as internal drafts. The current working position is that credits are service entitlements only and require legal/accounting review before external reliance.
