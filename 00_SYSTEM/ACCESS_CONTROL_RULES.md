# Access Control Rules

## Default Rule

Treat files as internal unless explicitly classified otherwise.

## Investor Sharing

Only share files classified as `investor_safe` or `public`.

Before sharing, confirm that the file is current, supported by evidence, and does not expose restricted internal logic, personal data, sensitive legal risk, or sensitive financial detail.

## Restricted Materials

Files classified as `restricted_internal`, `legal_sensitive`, `financial_sensitive`, or `personal_data` should not be placed in investor-facing folders without a redacted version.
