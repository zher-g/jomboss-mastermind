## 1. Purpose

This checklist is the master control document for building and maintaining the JomBoss investor data room.
The goal is to make JomBoss due-diligence ready from an early stage by organising all key corporate,
financial, legal, operational, product, customer, and fundraising materials in one structured system.
The data room should help JomBoss answer investor, accelerator, grant, and partner questions with
evidence.

## 2. Data Room Standard

A document is considered data-room ready only if it is:
Accurate
Current
Properly named
Stored in the correct folder
Classified by confidentiality level
Supported by source evidence where needed
Free from obvious contradictions
Clearly marked if draft or unapproved
